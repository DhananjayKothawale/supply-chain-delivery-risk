# Supply Chain Delivery Risk Analysis

## Overview
This project analyzes delivery delay risks in a supply chain using Python, SQL, and Power BI. 
The goal is to identify structural drivers of delivery delays and provide actionable business recommendations.

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL (SQLite)
- Power BI
- Jupyter Notebook

## Project Workflow
1. Defined the business problem around delivery delays
2. Cleaned and prepared data using Python
3. Created analytical fact tables using SQL
4. Computed key KPIs and delay metrics
5. Built an interactive Power BI dashboard
6. Generated insights and business recommendations

## Key Insights
- ~57% of orders are delivered late
- Shipping mode is the primary delay driver
- First Class shipping consistently violates SLA
- Standard Class is the most stable shipping option
